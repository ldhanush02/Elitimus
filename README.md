Logic Used for puzzle

The logic used to know the soft skill( cognitive flexibility ) is based on the Time taken to solve the word scramble puzzle and the Number of hints used by the users. Cognitive flexibility is the ability to shift one's thinking or attention between different tasks or ideas or to simultaneously consider multiple concepts or perspectives. As different Hints will reduce different points from the overall score which we take for 50% evaluation of soft skill ( cognitive flexibility ) and the remaining 50% depends upon time consumed by the user like if the user completed the puzzle in less than 10sec or 1 min etc. So In this way, I created a Logic of it.
